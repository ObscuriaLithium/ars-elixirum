# AI Usage Policy

This project was created as a result of deliberate human work – thinking, experimenting, making mistakes, and refining ideas over time. Because of that, I have a clear stance: I do not want this project to be used in connection with artificial intelligence systems in any form.

That includes feeding the source code, assets, or compiled files into AI tools, whether for analysis, code generation, modification, or even as reference material. It also includes using any part of this project in datasets, training pipelines, or evaluation processes for machine learning models. In simple terms, if an AI system is involved in any meaningful way, I do not want this project to be part of that interaction.

As both a developer and an artist, I believe that creative and technical work should come from people, not be reconstructed or extended through automated systems trained on the work of others. I don't want my projects to contribute to that process, directly or indirectly.

If you are using this project, I'm asking you to respect that position. Even if something is technically possible, that doesn’t mean it aligns with the spirit in which this work was shared.

Thank you for understanding.
